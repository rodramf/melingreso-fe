Algunas consideraciones:
  • Para los cálculos de linealidad entre planetas se consideró un factor de error debido al redondeo de decimales que se realizan en el cálculo de posición en los ejes X e Y.
  • Para determinar si el Sol se encuentra dentro del triangulo formado por los planetas se utilizó las fórmulas de este blog
  • Las variables generales del sistema y de los planetas son configurables desde el front end desde la sección de Settings
Arquitectura de solución
  • El desarrollo sigue el modelo API Driven Application, con el objetivo de independizar la lógica de negocio de los distintos tipos de front end (web, mobile, smart tv, etc)
  • Se utilizó Grails 2.4.5 para el desarrollo del back-end
  • Base de Datos in-memory H2
  • La capa de Front End es Singel Page App en Angular 1.3
  • Las APIs son REST Full para todos los casos
  • Dejo disponible el export del cliente POSTMAN de chrome para que puedan consultar la documentación de las APIs generadas
  • No se consideró para esta versión aspectos de seguridad ni manejo de sesión de usuario
  • Tampoco se incluyeron el desarrollo de las pruebas unitarias y de integración
